七成毕业生月薪不足6000：除了工资，第一份工作还应该看重什么？

Posted on 2019-07-05 | | Visitors:

原文地址 https://mp.weixin.qq.com/s/0dn2DnEJT1T7ajr2BPYG8w

昨天关于毕业生薪酬的数据上了微博热搜，戳中了很多年轻人的痛点：七成毕业生月薪不足6000。

根据《2019年大学生求职指南》数据显示，从实际签约月薪来看，有七成应届毕业生的签约月薪在6000元以内，4000元以下占比为35.77%。

毕业生薪酬不高带来的是签约难。应届生尚未签约的原因中，“薪水/待遇未达到期望值”仍是首要因素，比例达到38.23%。

毕竟是人生的第一份工作，很多毕业生都心怀期望慎之又慎，可是有时候，走走停停找来找去，却发现现实远比想象更残酷。一边是需求降低，一边是薪资减少，很多年轻人忍不住会纠结，第一份工作到底要不要在意薪酬。

年轻人到底要不要在意薪酬，其实不能一概而论，毕竟比起薪水，还有很多其他方面值得考虑。

1

首先，要考虑行业发展。

数据显示，在所有的招聘行业中，就业形势最好的行业依然是中介服务，今年该行业的企业招聘需求同比增长25%。

需求上涨的同时，在这个行业求职的大学生数量却在减少，2019届应届生投递中介服务行业的人数同比减少21%。

很多毕业生在职业选择上喜欢随大流，盲目涌向热门行业。

但是衡量一个行业，热门与否不应该是主要考量因素。有些热门行业本身并不具备长期发展，比如马云曾经举的一个例子，数据分析是热门职业，但是十年后，可能会被AI取代。

而且由于竞争激烈，人才辈出，你能谈判的资本并不多。反倒是冷门行业，由于招聘需求大，年轻人更容易谈出好条件。

宁向东曾经做过这样的比喻，选择行业其实就等于是选择投资。

当年，有一家投资机构，他们在选择是否要投资高铁行业的时候，首先问自己的就是，高铁在中国有没有前途？

选行业，其实也可以参考这个思路。选择一家公司，要看行业发展。它们做的事，是不是符合未来发展需要，只有在人们的生活息息相关的行业里，你才会找到未来事业发展的机会和希望。

2

其次，要考虑个人职业发展空间。

所谓个人职业发展空间，其实就是成长性。

我身边有两个很典型的例子。同学A毕业以后去了一家国企，月收入1万。但工作内容一成不变，两年以后，工资依旧是1万，而且可预见到未来涨幅很小。

同学B毕业后去了一家互联网公司，月入只有7000。但公司成长飞速，每年手里的奖金加起来，相当于16个月左右的工资。这两个人的年收入相差不多，而且看起来B同学的工作难度更大，因为奖金完全按照他的能力来。

但两年后，你会发现这两个人在市场的价值完全不一样。同学A想跳槽，屡屡碰壁，国企的安逸成了他简历上的缺陷。这也是为什么发展比起点重要。

可喜的是，很多年轻人看懂了这点。数据显示，大学生求职最关注的因素是，能够学习新东西，占比达到54%，其次才是赚钱。

而且年轻人在选择公司的时候不再盲目海投。在参与调研的2019应届毕业生中，超过一半的毕业生投递简历数量少于10份，投递简历数11份及以上的合计占比39.48%，和去年同期相比下降29.58%。

大学生在求职时选择“海投”行为的人数下降，正是说明他们在选择上更为谨慎。

一个人的收入=工资+未来可能性。所以，从职业发展角度来看，发展空间要优先考虑。

这不是说薪酬不重要，真正的好平台，反而不会榨干你现有的资源，几年之后抛弃你，而是会想着如何提高员工创造价值的动力和效能，让员工和公司一起增值。

3

最后，要保持更开放的心态。

过去，很多毕业生都说第一份工作一定要去大公司。现在大家似乎都有了更开放的心态，对创业公司的热情也开始高涨。

创业公司一直被认为辛苦且工作压力大，但调研结果显示，45.24%的应届毕业生表示会接受创业公司的offer，43.28%的高校毕业生持观望态度，并表示要看创业公司的行业或者领域而定。

对创业公司的接受度提升在我看来，是件好事。在选择接受创业公司的人群中，30.27%是因为创业公司能锻炼人。所以，现在的年轻人并不怕吃苦，他们在寻找更适合自己的平台和机会。

参与一家刚刚起步的公司，往往意味着有更大的机会接触到方方面面的知识。比如说做互联网金融的，可以接触到金融中的资产、风控、私人理财规划师等上下游资源，相比较在大公司做螺丝钉而言，你对业务可以接触的更加广泛，信息渠道也更加广泛，得到的发展锻炼也会更多。

而且，比起在大公司严格的晋升机制，创业公司的晋升没有定律。公司小，和老板之间距离更短，你做出的成绩很容易被看到。如果在职位上表现优异，相比较大公司而言，小公司工作，往往更容易得到越级晋升。

今天这个时代，创业公司遍地都是，更有自媒体的崛起，其实对于年轻人来说，除了国企公务员等铁饭碗，互联网的发展，带来的是更加丰富的职业选择。

说这些，并不是说第一份工作薪酬不重要，而是还有一些因素也很重要，他们决定你未来到底值多少钱。年轻人更应该以更开放的心态面对职业发展，综合考虑。

人生的第一份工作就是一个人的职业起点，这个起点高低与否其实没那么重要，重要的是，你要选择一条正确的赛道，长久地坚持，才能跑赢人生这场比赛。
