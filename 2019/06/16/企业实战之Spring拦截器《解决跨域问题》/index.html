跨域是什么？

跨域访问，简单来说就是 A 网站的 javascript 代码试图访问 B 网站时，浏览器出于安全考虑，自动阻止了对跨域服务的访问(包括对后端数据的增删改查请求)，这里浏览器统一遵循了一种策略，这个策略就是同源策略，同源策略也是浏览器最核心、最基本的安全功能。
