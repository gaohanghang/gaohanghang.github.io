еҸӮиҖғиө„ж–ҷпјҡ

гҖҠж·ұе…ҘзҗҶи§ЈJavaиҷҡжӢҹжңәгҖӢ

гҖҠHotspotе®һжҲҳгҖӢ

дёҖдёӘйқўиҜ•е®ҳеҜ№GCй—®йўҳзҡ„еҲҶжһҗ

зҗҶи§ЈJavaеһғеңҫеӣһж”¶жңәеҲ¶

JvmеҶ…еӯҳжЁЎеһӢ

еӣҫи§ЈJava еһғеңҫеӣһж”¶жңәеҲ¶

0. еәҸ

Java еһғеңҫеӣһж”¶жңәеҲ¶иҰҒиҖғиҷ‘зҡ„й—®йўҳеҫҲеӨҚжқӮпјҢжң¬ж–Үе°Ҷйҳҗиҝ°е…¶дёүдёӘж ёеҝғй—®йўҳпјҢеҢ…жӢ¬пјҡ

  • What е“ӘдәӣеҶ…еӯҳйңҖиҰҒеӣһж”¶пјҹ

    еҲӨж–ӯеҜ№иұЎжҳҜеҗҰеҸҜд»Ҙиў«еӣһж”¶зҡ„дёӨз§Қз»Ҹе…ёз®—жі•:

    1. еј•з”Ёи®Ўж•°жі•
    2. еҸҜиҫҫжҖ§еҲҶжһҗз®—жі•

  • When д»Җд№Ҳж—¶еҖҷеӣһж”¶пјҹпјҲе Ҷзҡ„ж–°з”ҹд»ЈгҖҒиҖҒе№ҙд»ЈгҖҒж°ёд№…д»Јзҡ„еһғеңҫеӣһж”¶ж—¶жңәпјҢMinorGC е’Ң FullGCпјү

  • How еҰӮдҪ•еӣһж”¶пјҹ(дёүз§Қз»Ҹе…ёеһғеңҫеӣһж”¶з®—жі•(ж Үи®°жё…йҷӨз®—жі•гҖҒеӨҚеҲ¶з®—жі•гҖҒж Үи®°ж•ҙзҗҶз®—жі•)еҸҠеҲҶд»Јж”¶йӣҶз®—жі• е’Ң дёғз§Қеһғеңҫж”¶йӣҶеҷЁ)

1. йҰ–е…ҲеӣһйЎҫJavaеҶ…еӯҳжЁЎеһӢ

иҝҗиЎҢж—¶еҶ…еӯҳжЁЎеһӢпјҢеҲҶдёәзәҝзЁӢз§Ғжңүе’Ңе…ұдә«ж•°жҚ®еҢәдёӨеӨ§зұ»пјҢе…¶дёӯзәҝзЁӢз§Ғжңүзҡ„ж•°жҚ®еҢәеҢ…еҗ«зЁӢеәҸи®Ўж•°еҷЁгҖҒиҷҡжӢҹжңәж ҲгҖҒжң¬ең°ж–№жі•ж ҲпјҢжүҖжңүзәҝзЁӢе…ұдә«зҡ„ж•°жҚ®еҢәеҢ…еҗ«Javaе ҶгҖҒж–№жі•еҢәпјҢеңЁж–№жі•еҢәеҶ…жңүдёҖдёӘеёёйҮҸжұ гҖӮ

пјҲ1пјүзәҝзЁӢз§ҒжңүеҢәпјҡ

  • зЁӢеәҸи®Ўж•°еҷЁпјҡи®°еҪ•жӯЈеңЁжү§иЎҢзҡ„иҷҡжӢҹжңәеӯ—иҠӮз Ғзҡ„ең°еқҖпјӣ
  • иҷҡжӢҹжңәж Ҳпјҡж–№жі•жү§иЎҢзҡ„еҶ…еӯҳеҢәпјҢжҜҸдёӘж–№жі•жү§иЎҢж—¶дјҡеңЁиҷҡжӢҹжңәж ҲдёӯеҲӣе»әж Ҳеё§пјӣ
  • жң¬ең°ж–№жі•ж ҲпјҡиҷҡжӢҹжңәзҡ„Nativeж–№жі•жү§иЎҢзҡ„еҶ…еӯҳеҢәпјӣ

пјҲ2пјүзәҝзЁӢе…ұдә«еҢәпјҡ

  • Javaе ҶпјҡеҜ№иұЎеҲҶй…ҚеҶ…еӯҳзҡ„еҢәеҹҹпјӣ
  • ж–№жі•еҢәпјҡеӯҳж”ҫзұ»дҝЎжҒҜгҖҒеёёйҮҸгҖҒйқҷжҖҒеҸҳйҮҸгҖҒзј–иҜ‘еҷЁзј–иҜ‘еҗҺзҡ„д»Јз Ғзӯүж•°жҚ®пјӣ
    • еёёйҮҸжұ пјҡеӯҳж”ҫзј–иҜ‘еҷЁз”ҹжҲҗзҡ„еҗ„з§Қеӯ—йқўйҮҸе’Ңз¬ҰеҸ·еј•з”ЁпјҢжҳҜж–№жі•еҢәзҡ„дёҖйғЁеҲҶгҖӮ

2. й—®йўҳдёҖпјҡWhat е“ӘдәӣеҶ…еӯҳйңҖиҰҒеӣһж”¶

еҸҜд»ҘйҖҡиҝҮдёӨз§Қз®—жі•жқҘеҲӨж–ӯеҜ№иұЎжҳҜеҗҰеҸҜд»Ҙиў«еӣһж”¶

  1. еј•з”Ёи®Ўж•°з®—жі•пјҡеҲӨж–ӯеҜ№иұЎзҡ„еј•з”Ёж•°йҮҸ
  2. еҸҜиҫҫжҖ§еҲҶжһҗз®—жі•пјҡеҲӨж–ӯеҜ№иұЎзҡ„еј•з”Ёй“ҫжҳҜеҗҰеҸҜиҫҫ

еј•з”Ёи®Ўж•°з®—жі•

е Ҷдёӯзҡ„жҜҸдёӘеҜ№иұЎе®һдҫӢйғҪжңүдёҖдёӘеј•з”Ёи®Ўж•°гҖӮеҪ“дёҖдёӘеҜ№иұЎиў«еҲӣе»әж—¶пјҢдё”е°ҶиҜҘеҜ№иұЎе®һдҫӢеҲҶй…Қз»ҷдёҖдёӘеј•з”ЁеҸҳйҮҸпјҢиҜҘеҜ№иұЎе®һдҫӢзҡ„еј•з”Ёи®Ўж•°и®ҫзҪ®дёә 1гҖӮеҪ“д»»дҪ•е…¶е®ғеҸҳйҮҸиў«иөӢеҖјдёәиҝҷдёӘеҜ№иұЎзҡ„еј•з”Ёж—¶пјҢеҜ№иұЎе®һдҫӢзҡ„еј•з”Ёи®Ўж•°еҠ  1пјҲa = bпјҢеҲҷbеј•з”Ёзҡ„еҜ№иұЎе®һдҫӢзҡ„и®Ўж•°еҷЁеҠ  1пјүпјҢдҪҶеҪ“дёҖдёӘеҜ№иұЎе®һдҫӢзҡ„жҹҗдёӘеј•з”Ёи¶…иҝҮдәҶз”ҹе‘Ҫе‘ЁжңҹжҲ–иҖ…иў«и®ҫзҪ®дёәдёҖдёӘж–°еҖјж—¶пјҢеҜ№иұЎе®һдҫӢзҡ„еј•з”Ёи®Ўж•°еҮҸ 1гҖӮзү№еҲ«ең°пјҢеҪ“дёҖдёӘеҜ№иұЎе®һдҫӢиў«еһғеңҫж”¶йӣҶж—¶пјҢе®ғеј•з”Ёзҡ„д»»дҪ•еҜ№иұЎе®һдҫӢзҡ„еј•з”Ёи®Ўж•°еҷЁеқҮеҮҸ 1гҖӮд»»дҪ•еј•з”Ёи®Ўж•°дёә0зҡ„еҜ№иұЎе®һдҫӢеҸҜд»Ҙиў«еҪ“дҪңеһғеңҫж”¶йӣҶгҖӮ

зјәзӮ№пјҡеҫҲйҡҫи§ЈеҶіеҜ№иұЎд№Ӣй—ҙзҡ„зӣёдә’еҫӘзҺҜеј•з”Ёзҡ„й—®йўҳгҖӮ

еҸҜиҫҫжҖ§еҲҶжһҗз®—жі•

еҸҜиҫҫжҖ§еҲҶжһҗз®—жі•жҳҜйҖҡиҝҮеҲӨж–ӯеҜ№иұЎзҡ„еј•з”Ёй“ҫжҳҜеҗҰеҸҜиҫҫжқҘеҶіе®ҡеҜ№иұЎжҳҜеҗҰеҸҜд»Ҙиў«еӣһж”¶гҖӮ

зЁӢеәҸжҠҠжүҖжңүзҡ„еј•з”Ёе…ізі»зңӢдҪңдёҖеј еӣҫпјҢйҖҡиҝҮдёҖзі»еҲ—зҡ„еҗҚдёә вҖңGC RootsвҖқ зҡ„еҜ№иұЎдҪңдёәиө·е§ӢзӮ№пјҢд»ҺиҝҷдәӣиҠӮзӮ№ејҖе§Ӣеҗ‘дёӢжҗңзҙўпјҢжҗңзҙўжүҖиө°иҝҮзҡ„и·Ҝеҫ„з§°дёәеј•з”Ёй“ҫпјҲReference ChainпјүгҖӮеҪ“дёҖдёӘеҜ№иұЎеҲ° GC Roots жІЎжңүд»»дҪ•еј•з”Ёй“ҫзӣёиҝһпјҲз”Ёеӣҫи®әзҡ„иҜқжқҘиҜҙе°ұжҳҜд»Һ GC Roots еҲ°иҝҷдёӘеҜ№иұЎдёҚеҸҜиҫҫпјүж—¶пјҢеҲҷиҜҒжҳҺжӯӨеҜ№иұЎжҳҜдёҚеҸҜз”Ёзҡ„пјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮеңЁJavaдёӯпјҢеҸҜдҪңдёә GC Root зҡ„еҜ№иұЎеҢ…жӢ¬д»ҘдёӢеҮ з§Қпјҡ

  • иҷҡжӢҹжңәж Ҳ(ж Ҳеё§дёӯзҡ„еұҖйғЁеҸҳйҮҸиЎЁ)дёӯеј•з”Ёзҡ„еҜ№иұЎпјӣ
  • ж–№жі•еҢәдёӯзұ»йқҷжҖҒеұһжҖ§еј•з”Ёзҡ„еҜ№иұЎпјӣ
  • ж–№жі•еҢәдёӯеёёйҮҸеј•з”Ёзҡ„еҜ№иұЎпјӣ
  • жң¬ең°ж–№жі•ж ҲдёӯNativeж–№жі•еј•з”Ёзҡ„еҜ№иұЎпјӣ
