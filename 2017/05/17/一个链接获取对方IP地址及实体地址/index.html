一个链接获取对方IP地址及实体地址

Posted on 2017-05-17 | | Visitors:

主要用到一个网址：https://www.dingwei.tv/

1、打开后，在左侧会显示自己的信息
（如果打开后是一片黑色，请更换浏览器)

2、在右上角处，点击生成探测地址

3、使用此功能需要安装 探测地址扩展插件，点击下载

4、把插件安装到浏览器中

5、再次打开 www.dingwei.tv 页面 ，在右上角就会展现出带有自己唯一标识的探测地址

6、把探测地址发给对方，只要对方打开后，你这里就会显示打开者相关信息
（相关信息包括：IP、地址、2D地图、实景地图、分辨率、浏览器、访客系统、网络接入商）

链接地址：https://www.dingwei.tv/
