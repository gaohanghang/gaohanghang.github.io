Redis

1. 什么是Redis

  1. Redis是一种key-value内存数据库
    • 底层算法就是散列表
  2. 性能极佳，查询110000次/秒，写80000次/秒
  3. 提供了持久化支持。
  4. Redis的Value支持5种数据类型，使用方便。

Redis查询性能非常优秀：在软件中经常做为数据库的缓存，提供软件的性能，也就是用于解决“高性能”问题。
